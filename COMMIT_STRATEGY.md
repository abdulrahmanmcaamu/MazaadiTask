# Natural Git Commit Strategy

## Commit Order (Step by Step Development)

1. **Project Setup & Dependencies**
   - Add Hilt, Apollo, Navigation dependencies
   - Update build configurations

2. **Clean Architecture Foundation**
   - Create folder structure
   - Domain models

3. **Domain Layer Implementation**
   - Repository interface
   - Use cases

4. **Data Layer - GraphQL Setup**
   - GraphQL queries
   - Apollo client configuration

5. **Data Layer - Repository Implementation**
   - Remote data source
   - Repository implementation

6. **Dependency Injection Setup**
   - Hilt modules
   - Application class

7. **MVI Architecture - Launches Screen**
   - State, Intent, ViewModel

8. **UI Implementation - Launches Screen**
   - Compose UI with pagination

9. **MVI Architecture - Launch Detail Screen**
   - State, Intent, ViewModel

10. **UI Implementation - Launch Detail Screen**
    - Compose UI with mission patch

11. **Navigation Setup**
    - NavGraph implementation

12. **Testing**
    - Unit tests
    - Instrumented tests

13. **Final Polish**
    - Error handling improvements
    - Code comments and documentation
    - ProGuard rules
